### Installation & Setup

1. **Clone the repository:**

   ```
   git clone <Your Repository URL>
   cd personal-page-cv
   ```

3.  **Install Dependencies:**
    
    ```
    npm install
    ``` 
    
4.  **Run the Application:**
    
   
    ```
    npm start
    ```

    
    Open your web browser and visit http://localhost:4000.
